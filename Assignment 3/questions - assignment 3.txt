1. What is the difference between position: absolute and position: fixed ? 
	- Position: absolute means the element is positioned relative to its first positioned (not 	static) ancestor element. Position: fixed means the element is positioned relative to the 	browser window.

2. With CSS3, how do we access the third element within another element?
	- We can access the third element within another element by using the selector 			element>:last-child>:last-child.
